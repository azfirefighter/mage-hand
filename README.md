#Magehand
A real-time campaign and character management tool for table-top roleplaying games.

## Usage
- Install: `npm install`
- Step 1: Start server with `DEBUG=myapp:* npm start`
- Step 2: Also have `webpack -w` running
- Step 3: Profit?
